== 機能 ==
★ですます調に変更してください。
マシン語関数を宣言

== 書式 ==

★ 書式に省略可能な繰り返し [,引数...] が含まれている場合は、{,引数} に変更してください。
DEF USR [ 番号 ] =アドレス

== 解説 ==

★なるべく、ですます調に変更してください。分量が多い場合はそのままで構いません。


アドレスから始まるマシン語ルーチンを関数として呼び出せるよう宣言する。番号は0～9。省略時は0。 宣言後はUSER関数で呼び出せる。

マシン語ルーチンには、引数が次のようにレジスタに渡される。
; A
: 引数の型。2: 整数、3: 文字列、4: 単精度実数、8: 倍精度実数
; DE
: (文字列の場合)文字列の長さと格納先を示したテーブルのアドレス(VARPTRが文字列変数に対して返すのと同じ)
; HL
: (数値の場合)値の格納先のアドレス。ただし、倍精度の場合は格納先アドレス+4。格納先サイズは整数、単精度、倍精度それぞれ2、4、8バイト。

BASICに値を返すには、引数として受け取った領域に値を書き出して、RETを実行する。返す値の型は引数と同じとなる。

マシン語ルーチンに用意されたスタックは16バイト。それ以上必要な場合は独自に用意すること。
