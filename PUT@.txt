== 機能 ==
★ですます調に変更してください。
配列に読み込まれた矩形領域を画面に描画

== 書式 ==

★ 書式に省略可能な繰り返し [,引数...] が含まれている場合は、{,引数} に変更してください。
PUT [ @ ] ( X , Y ) , 配列変数名 [ ( 要素 ) ] [ , 条件 ] [ , 描画色 , 背景色 ]

== 解説 ==

★なるべく、ですます調に変更してください。分量が多い場合はそのままで構いません。


GET@で配列に読み込んだ矩形領域をグラフィック画面に描画する。 座標はスクリーン座標系(VIEWを参照)で指定する。

GET@で読み込み先に指定した配列変数名を指定する。配列の先頭の要素以外に読み込んだ場合は要素も指定する。

条件は、描画時に適用する演算を次の中から指定。省略時はXOR。
; PSET
: そのまま描画。
; PRESET
: 色を反転して描画。カラーモードの場合は「7-元のパレット番号」の結果を描画。白黒モードの場合は白と黒を入れ替え。
; OR
: 描画先の画面の内容との間でOR演算をした結果を描画。
; AND
: 同様に、AND演算。
; XOR
: 同様に、XOR演算。

描画色、背景色は、白黒モードで読みこんだ矩形領域をカラーモードで描画する場合に、「白」「黒」をそれぞれどのパレット番号で描画するかを指定する。 それ以外の場合は指定しない。

PUT@は、@を省略し単にPUTとしてよい。
