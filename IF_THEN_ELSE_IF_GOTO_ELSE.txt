; 書式1
: IF 条件 THEN 真の場合の処理 [ ELSE 偽の場合の処理 ]
; 書式2
: IF 条件 GOTO 真の場合の行番号 [ ELSE 偽の場合の処理 ]

== 解説 ==

★なるべく、ですます調に変更してください。分量が多い場合はそのままで構いません。


条件の真偽に応じて真の場合の処理または偽の場合の処理を実行する。偽の場合の処理が省略された場合は次の行に進む。条件(数値)は0以外が真、0が偽。

真の場合の処理、偽の場合の処理はそれぞれ次のとおり記述する。
*文を記述する。コロン(<code>:</code>)区切りで複数の文を記述可能。
*IFも記述可。
*単に行番号のみ記述すると、その行へジャンプする。
