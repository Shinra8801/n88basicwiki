== 機能 ==
★ですます調に変更してください。
キーボードから値を入力

== 書式 ==
<!--

  ・[A]    Aを省略可
  ・<A|B>  AまたはBを指定

★ 書式に省略可能な繰り返し [,引数...] が含まれている場合は、{,引数} に変更してください。
; 書式1
: INPUT [ プロンプト ; ] 変数名 [ , 変数名... ]
; 書式2
: INPUT WAIT待ち時間 , [ プロンプト ; ] 変数名 [ , 変数名... ]

== 解説 ==

★なるべく、ですます調に変更してください。分量が多い場合はそのままで構いません。


キーボードから値を入力し指定の変数に代入する。複数の値を入力する場合は、その個数だけ変数名を指定する。 実行時は次のようにキー入力する。
*各値をカンマ(<code>,</code>)で区切って入力する。
*カンマを含む文字列、空白で始まる(または終わる)文字列を入力する場合は、ダブルクォーテーション(<code>"</code>)で囲む。ダブルクォーテーション自体は入力できない。
*最後にリターンキーを押す。

プロンプト(文字列定数)が指定された場合は、入力に先立ちプロンプトと<code>?</code>、半角空白が表示される。プロンプト直後の区切り文字に<code>;</code>ではなく<code>,</code>を使用した場合は、<code>?</code>、半角空白は表示されない。

書式2は、時間制限付きで入力する。待ち時間(単位は0.1秒)の間に入力されない場合は次の行に実行が移る。 コロン(<code>:</code>)で他の命令を続けている場合はそれらもスキップされる。
