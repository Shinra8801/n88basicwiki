== 機能 ==
★ですます調に変更してください。
書式指定付きで画面へ表示

== 書式 ==
<!-- 
  ヘルプ ( http://www.cyross.com/pc8801wiki/index.php/%E3%83%98%E3%83%AB%E3%83%97 ) を参照。
  ・[A]    Aを省略可
  ・<A|B>  AまたはBを指定
  ・{A}    Aを繰り返し指定可能。省略も可。
-->

★ 書式に省略可能な繰り返し [,引数...] が含まれている場合は、{,引数} に変更してください。
PRINT USING書式 ; 値 [ , 値... ]

== 解説 ==

★なるべく、ですます調に変更してください。分量が多い場合はそのままで構いません。


値を書式にしたがってフォーマットし表示する。 複数の値を表示する場合はカンマ(<code>,</code>)またはセミコロン(<code>;</code>)で区切る。 PRINT USINGの最後がカンマやセミコロンで終わる場合は改行しない。書式中に以下の制御文字列が現れると、その部分は値に置換されて表示される。n番目の制御文字列はn番目の値で置換される。値の数が制御文字列より多い場合は書式の先頭から再利用される。
*文字の制御文字列
*; <code>@</code>
*: 文字列を表示
*; <code>!</code>
*: 文字列の最初の1文字のみ表示。2バイト文字には利用できない。
*; <code>&　　&</code>
*: &とそれに囲まれた半角スペースの部分に文字列を表示。表示しようとする文字列のほうが長い場合、余剰分は表示されない。
*数値の制御文字列(以下の文字を連結して1つの制御文字列を構成)
*; <code>#</code>
*: 数値を表示。#の個数分の桁数で右詰で表示。(例:<code>###</code> ⇒ 整数部のみ3桁表示)
*; <code>.</code>(ピリオド)
*: 小数点の位置を指定。(例:<code>###.##</code> ⇒ 整数部3桁と小数点以下2桁を表示)
*; <code>,</code>(カンマ)
*: 3桁ごとにカンマで区切ることを指定。#の並びの中のどこに置いてもよい。カンマ含めて桁数に数えられる。 (例:<code>#######,#</code>⇒ カンマ含めて9桁で表示。「1,234,567」がちょうど表示できる。)
*; <code>+</code>
*: 符号(+または-)の位置を指定。書式文字列の先頭または最後に付加。(例:<code>+###</code>)
*; <code>-</code>
*: 符号(半角空白または-)を値の後に表示することを指定。書式文字列の最後に付加。(例:<code>###-</code>)
*; <code>**</code>
*: 桁数に満たない場合に空白でなく"*"で埋めることを指定。#の並びの先頭に付加。**も桁数に数えられる。(例:<code>**#</code> ⇒ 整数部のみ3桁表示)
*; <code>\\</code>
*: 先頭に"\"を表示する。#の並びの先頭に付加。\\も桁数に数えられる。(例:<code>\\##</code> ⇒ 4桁(\含む)で表示
*; <code>**\</code>
*: **と\\の両方の機能を指定。
*; <code>^^^^</code>
*: 指数形式表示を指定。#の並びの後に指定。
*文字・数値共通
*; <code>_</code>
*: 上記の文字を制御文字として扱わずに通常の文字として表示することを指定。

日本語BASICの場合、2バイト文字のうち下位バイトが"@"、"\"、"^"、"-"と一致する文字は書式中に含められない。

数値の場合、整数部が指定の桁数で収まらない場合、先頭に"%"が表示される。小数点以下が指定の桁数に収まらない場合は収まるように四捨五入される。
