== 機能 ==
★ですます調に変更してください。
コメントを記述

== 書式 ==
<!--

  ・[A]    Aを省略可
  ・<A|B>  AまたはBを指定

★ 書式に省略可能な繰り返し [,引数...] が含まれている場合は、{,引数} に変更してください。
; 書式1
: REM [ コメント ]
; 書式2
: ' [ コメント ]

== 解説 ==

★なるべく、ですます調に変更してください。分量が多い場合はそのままで構いません。


コメントを記述する。REM以降、行の最後までコロン(<code>:</code>)も含めコメントと解釈され実行されない。 REMはアポストロフィ(書式2)で代替できる。この場合は、前の文に続ける場合でもコロン(<code>:</code>)を省略できる。

文字コード &HE9 を含む文字(♥、秘など)をコメントに記述する場合は書式2は利用できない。
<pre>
10 REM Hello world sample
20 PRINT "Hello N88-BASIC!" 'ﾒｯｾｰｼﾞ ｦ ﾋｮｳｼﾞｼﾏｽ
</pre>
