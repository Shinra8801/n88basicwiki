== 機能 ==
★ですます調に変更してください。
ファイル・ディスクの属性を設定

== 書式 ==
<!-- 
  ヘルプ ( http://www.cyross.com/pc8801wiki/index.php/%E3%83%98%E3%83%AB%E3%83%97 ) を参照。
  ・[A]    Aを省略可
  ・<A|B>  AまたはBを指定
  ・{A}    Aを繰り返し指定可能。省略も可。
-->

★ 書式に省略可能な繰り返し [,引数...] が含まれている場合は、{,引数} に変更してください。
; 書式1
: SET ドライブ番号 , 属性文字
; 書式2
: SET ファイルディスクリプタ , 属性文字
; 書式3
: SET #ファイル番号 , 属性文字

== 解説 ==

★なるべく、ですます調に変更してください。分量が多い場合はそのままで構いません。


指定のフロッピーディスクやファイルに属性を付ける。属性は文字列で、以下の中から指定する。
; <code>"P"</code>
: 書き込み禁止。書き込みと削除が不可能になる。
; <code>"R"</code>
: 書き込み確認。データの書き込み時は、直後に読み出しメモリの内容と比較することで正常に書き込みまれたかを確認する。
; <code>その他</code>
: すべての属性を解除する。

書式1は、ドライブ番号(1～)のフロッピーに対して、書式2, 3は個別のファイルに対して属性を設定する。 書式3は実行するとそれ以降のそのファイルへの書き込みに対して属性が機能する。
:<code>SET "1:bitinn.his","P"</code> ⇒ ドライブ1の"bitinn.his"を書き込み禁止に。
