== 機能 ==
★ですます調に変更してください。
直線・矩形を描画

== 書式 ==
<!-- 
  ヘルプ ( http://www.cyross.com/pc8801wiki/index.php/%E3%83%98%E3%83%AB%E3%83%97 ) を参照。
  ・[A]    Aを省略可
  ・<A|B>  AまたはBを指定
  ・{A}    Aを繰り返し指定可能。省略も可。
-->

★ 書式に省略可能な繰り返し [,引数...] が含まれている場合は、{,引数} に変更してください。
LINE [ ( X1 , Y1 ) ] - ( X2 , Y2 ) [ , 描画色 ] [ , オプション ] [ , スタイル ]

== 解説 ==

★なるべく、ですます調に変更してください。分量が多い場合はそのままで構いません。


オプションに応じて、次のとおり直線または矩形を描画する。
; 省略
: 始点と終点を結ぶ直線を描画
; B
: 始点-終点を対角線とする矩形を描画
; BF
: 始点-終点を対角線とする矩形を描画し、中を塗りつぶす

始点(X1,Y1)、終点(X2,Y2)はワールド座標系で指定する。 STEP(X,Y)の形式(相対座標)で指定してもよい(POINT(最終描画位置設定)参照)。 始点を省略した場合、LP(最終描画位置)を始点とする。

描画色はパレット番号で指定。省略時はCOLORで指定した描画色。

スタイルは、破線を描画したい場合に指定する。16 bitの整数で表され、16ドット分の線について、描画の有無を指定する。17ドット目以降は1ドット目からの指定が繰り返して適用される。 BFオプション指定時はスタイルは指定できない。
; bit 0
: 1ドット目 1:描画する、0:描画しない
; bit 1
: 2ドット目
; ～
: 
; bit 15
: 16ドット目
:例:<code>&h18FF</code>(2進数表記で<code>0001100011111111</code>) ⇒ 1点鎖線
