== 機能 ==
★ですます調に変更してください。
BASICで使うメモリの設定と変数初期化

★ 書式に省略可能な繰り返し [,引数...] が含まれている場合は、{,引数} に変更してください。
CLEAR [ 文字列領域サイズ ] [ , メモリ上限 ] [ , スタックサイズ ]

== 解説 ==

★なるべく、ですます調に変更してください。分量が多い場合はそのままで構いません。


変数を初期化する。配列変数は削除される。DEF FN、DEF USR、DEFINT、DEFSNG、DEFDBL、DEFSTRによる定義も無効となる。
; 文字列領域サイズ
: 意味を持たない。
; メモリ上限
: BASICが利用するメモリの上限のアドレス。
; スタックサイズ
: BASICが利用するスタックのバイト数。デフォルトは512。FOR、GOSUB、PAINTなどで利用される。

メモリ上限+1から始まり、以下で表示されるアドレスまでのエリアは、マシン語プログラム等の配置場所として利用できる。
:<code>? HEX$(PEEK(&HE7E8)+PEEK(&HE7E9)*256)</code>
